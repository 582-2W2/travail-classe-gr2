Limiter les dimensions de l'image

Utiliser les propriétés width, height, min-width, min-height, max-width, max-height pour limiter les dimensions de l'image.

En effet, on ne peut pas toujours être sûr des dimensions réelles de l'image, surtout pour les pages Web dont le contenu est généré dynamiquement (à travers un gestionnaire de contenu par exemple).

Propriété object-fit: contain

Utiliser la propriété object-fit pour limiter l'image dans son conteneur, quitte à avoir des bandes horizontales/verticales vides.

Dans cet exemple, on tient à avoir la largeur et la hauteur de l'image à 100%, mais pour éviter d'étirer l'image, on utilise la propriété CSS object-fit.

Propriété object-fit: cover

Utiliser la propriété object-fit pour tout le conteneur avec l'image, quitte à rogner l'image verticalement ou horizontalement.

Dans cet exemple, on tient à avoir la largeur et la hauteur de l'image à 100%, mais pour éviter d'étirer l'image, on peut avoir recours à la propriété CSS object-fit.

Propriété object-position

Utiliser la propriété object-position pour spécifier le placement de l'image si la propriété object-fit: contain est utilisée, ou encore le point à partir duquel l'image sera rognée dans le cas de object-fit: cover.

Dans cet exemple, on veut que la barque blanche soit toujours en focus dans l'image, et on utilise donc la propriété object-position: bottom left.
